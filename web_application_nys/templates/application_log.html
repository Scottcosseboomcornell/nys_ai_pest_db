Application Log

This page will allow you to log and track pesticide applications for your crops.

Planned Features:

  • Application Planning: Create planned applications by selecting pesticides from the search
  • Date & Time: Schedule applications with specific dates and times
  • Rate Selection: Choose application rates based on pesticide labels
  • Calculations: Automatic calculation of product and water amounts based on your farm's acreage
  • REI Tracking: Track Re-entry Intervals (REI) for each application
  • PHI Tracking: Track Pre-Harvest Intervals (PHI) based on harvest dates
  • Notes: Add custom notes for each application
  • Export: Export application logs as Excel or PDF
  • Data Storage: All logs stored in Supabase and accessible across devices

This feature requires user authentication. Please set up Supabase to enable this functionality.
